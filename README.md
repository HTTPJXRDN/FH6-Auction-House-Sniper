# FH6 Auction House Sniper

## Automatic auction house sniper for Forza Horizon 6

Lightweight tool that automatically refreshes and buys out auctions at extremely high speed. It won't always win them
but if the server refresh is matched with the tool loop you should certainly win. 

---

# Features

- Fast auction refresh
- Automatic buyout
- Lightweight EXE
- Easy setup
- Designed for fast sniping

---

# Requirements

- Windows 10/11
- Fullscreen
- Focused on FH6 window while running
- Wired ethernet connection
- Disable moving background (display settings)
- Use Festival Site, not your house
