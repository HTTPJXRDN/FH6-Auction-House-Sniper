# FH6 Auction House Sniper

## Automatic auction house sniper for Forza Horizon 6

Lightweight tool that automatically refreshes and buys out auctions at extremely high speed. It won't always win them
but if the server refresh is matched with the tool loop you should certainly win. 

---

# Features

- Fast auction refresh
- Automatic buyout
- Lightweight EXE
- Easy setup
- Designed for fast sniping

---

# Requirements

- Windows 10/11
- Fullscreen
- Focused on FH6 window while running
- Wired ethernet connection
- SSD recommended (NVMe M.2 best)
- Disable moving background (in-game display settings)
- Use the Festival Site, not your home or estate
- (optional) you can set your graphics settings to very low and also get into a Peel P50 to try and make things faster, although on my end with Extreme preset at 4K I saw no difference.

## Step 4 — Launch the Sniper

Launch FH6 Auction Sniper and immediately click back into FH6 (you have 5 seconds to do so).

Move your mouse somewhere safe (ideally right above the logo in the bottom right) where it cannot click FH6 UI buttons accidentally.

-Note - This tool reproduces keyboard clicks, once you see the screen saying attempting buyout, you need to click the icon in the bottom right (circled in red) to close the tool. Otherwise, it will
get all wonky in the menus and possibly buy a random car in the aution house.
